# API Docs

## /user/singup (POST)

Create new user

- **Request**

```json
{
  "userName": "가나다@naver.com",
  "password": "1234
}
```

- **Response**

```json
{
  "user_name": "가나다",
  "user_email": "가나다@naver.com",
}
```
